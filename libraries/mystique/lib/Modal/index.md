## API
| 参数        | 说明                                     | 类型        | 默认值 |
|-----------  |----------------------------------------  | ---------- |-------|
|visable      | 是否显示                                  | boolean    | false |
|maskClosable | 点击遮罩是否关闭                          | boolean     | false |
|title        | 点击关闭的调用方法                        | function     | -     |
|onClose      | 层的名称                                 | function    | -     |
|className    | 最外层容器类名                            | string      | -     |
